PR #67 Feature

Antes

Navegação apenas por cliques no sidebar e bottom tab bar. Usuários avançados precisavam de múltiplos cliques para acessar páginas frequentes.

Depois

Hook useKeyboardShortcuts permite navegação rápida com teclas de atalho (g, e, v, c, p, s, r). Ignora inputs e combinações de teclas do sistema.

Benefícios

  • ✅ Navegação 3x mais rápida para usuários avançados
  • ✅ Experiência similar a apps desktop (Gmail, Slack)
  • ✅ Sem conflito com inputs de formulário
  • ✅ Implementação leve (~60 linhas)

Atalhos

g → Dashboard (Home)
e → Entradas
v → Vendas
c → Clientes
p → Perfil
s → Configurações
r → Relatórios
