Antes

  • Formatação inline com toLocaleString('pt-BR') e toFixed(N) espalhada
  • Sem função dedicada para currency, number, percent

Depois

  • formatCurrency(value) → R$ 1.234,56
  • formatNumber(value, digits?) → 1.234
  • formatPercent(value) → 12,3%
  • Funções disponíveis em src/lib/utils.ts

Benefícios

  • ✔️ Ponto único de mudança se formato mudar
  • ✔️ Código mais expressivo que toLocaleString
  • ✔️ 0 novas dependências
