🔒 Sprint #6 — Confirmações + Error Handling

PR #66 Tornar o app mais seguro e resiliente

✅ AlertDialog em Exclusões de Configuração
Exclusões agora pedem confirmação antes de executar
Antes: Botão "Excluir" executava direto, sem aviso. Risco de exclusão acidental.
Depois: AlertDialog com nome do item sendo excluído. Botão de confirmação em vermelho.
✅ Toast Feedback em Mutations
Feedback visual quando operações falham ou funcionam
Antes: Mutations falhavam silenciosamente. Usuário não sabia se funcionou.
Depois: Toast.error para falhas, Toast.success para sucesso. Mensagens amigáveis em pt-BR.
✅ Debug Log Estruturado
Sistema de logging para diagnosticar erros
Implementado: src/lib/logger.ts com logError() e logDestructiveOp(). Persiste no localStorage.
✅ Mensagens de Erro Amigáveis no Login
Erros do Supabase traduzidos para português
Antes: Mensagens técnicas como "Invalid login credentials".
Depois: "Credenciais inválidas", "Email já cadastrado", etc.

🛡️ Sprint #8 — Proteção contra Exclusão em Cascata

PR #68 Alinhar frontend com restrições do database

✅ Verificação de Relacionamentos
Sistema previne exclusão de dados vinculados
Comportamento:
  • Owner: Aviso quando possui contas vinculadas (cascade delete)
  • Program: Botão desabilitado se houver contas (restrict)
  • OrigemType: Botão desabilitado se houver entradas (restrict)

🎯 Sprint #9 — UX Improvements

PR #69 Melhorar experiência do usuário

✅ Modal de Ajuda para Atalhos
Usuários podem descobrir atalhos sem documentação externa
Implementado: Modal acessível por ? ou clicando no ícone de teclado no header.
✅ Toast de Sucesso em Operações
Feedback positivo após conclusão de operações destrutivas
Mensagens: "Conta excluída com sucesso", "Cliente excluído com sucesso", etc.

🌍 Sprint #10 — i18n (Internacionalização)

PR #70 Suporte a múltiplos idiomas

✅ Sistema de Internacionalização
Interface traduzível para múltiplos idiomas
Idiomas suportados: 🇧🇷 Português (padrão), 🇺🇸 English
✅ Toggle de Idioma
Usuários podem alternar idioma facilmente
Implementado: Bandeira no header que alterna entre pt-BR e en-US.
✅ Login com Traduções
Mensagens de erro traduzidas
Exemplos: "Credenciais inválidas", "Email já cadastrado", "Senha deve ter pelo menos 6 caracteres"
