Root Cause (Overflow)

Debug no bundle de produção:
  html.scrollWidth=433, html.clientWidth=393
  Elemento culpado: <div class="ml-auto flex items-center gap-2">
    width=280px (LanguageSelector 36 + gap 8 + KbdShortcuts 36 + gap 8 + GlobalSearch 192)
    left=153px → right edge = 433px

  Cálculo:
    Header: px-6 = 24px × 2 = 48px → 393 - 48 = 345px disponível
    Left: SidebarTrigger 28 + ml-4 16 + "MilesControl" ~90 = ~134px
    Right available: 345 - 134 ≈ 211px
    Right actual: 36 + 8 + 36 + 8 + 192 = 280px ❌ (overflow de 69px)

  Fix:
    Header: px-4 = 16px × 2 = 32px → 393 - 32 = 361px disponível
    Right available: 361 - 134 ≈ 227px
