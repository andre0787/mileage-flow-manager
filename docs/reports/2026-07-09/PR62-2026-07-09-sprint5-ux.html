Benefícios

  • ✅ Visibilidade: Entradas pendentes visíveis em todas as páginas (não só Dashboard)
  • ✅ Conversão: CTAs em empty states guiam usuário para ação (criar entrada/venda)
  • ✅ Performance: Listas paginadas evitam renderizar centenas de itens de uma vez
  • ✅ UX mobile: Badge no bottom tab bar funciona em telas pequenas
  • ✅ Manutenibilidade: Pagination.tsx reutilizável, props opcionais não quebram uso existente
  • ✅ Zero dependências: Tudo implementado com Tailwind + componentes existentes

Decisões de Design

  • Badge amber: Cor de alerta, sem ser intrusivo — consistente com o Dashboard existente
  • 20 itens/página: Balance entre visibilidade e performance — ajustável via constante
  • Paginação client-side: Dados já carregados via React Query — sem requests extras
  • Push notifications adiadas: Council decidiu que badge resolve 90% do caso de uso
